# UI 前端套件參考

精選各 UI 元件庫的官方 Demo 連結。開發前端或 UI/UX 時,先依專案框架與需求從下列分類挑選,可用 WebFetch 直接瀏覽官方 Demo 確認元件外觀、動畫與互動效果。

來源:https://wiki.david888.com/share/zx6xm9

- [Naive UI](https://www.naiveui.com/zh-CN/os-theme/components/switch)
- [Element Plus](https://element-plus.org/)
- [Vuetify](https://vuetifyjs.com/)
- [Ant Design Vue](https://antdv.com/)
- [PrimeVue](https://primevue.org/)
- [Nuxt UI](https://ui.nuxt.com/components)
- [TDesign Vue Next](https://tdesign.tencent.com/vue-next/)
- [Arco Design Vue](https://arco.design/vue/en-US/component/button)
- [Quasar](https://quasar.dev/vue-components/)
- [Varlet UI](https://varletjs.org/)

- [Reka UI](https://reka-ui.com/)

- [Motion Primitives](https://motion-primitives.com/)

## 依需求快速挑選

| 需求 | 優先看 |
|---|---|
| 最炫的動畫特效 | React Bits → Aceternity UI → Magic UI → Motion Primitives → SmoothUI |
| 精緻、適合產品的元件 | HeroUI → Mantine → Nuxt UI → Kumo → Radix Themes |
| AI 產品的新型元件 | Vercel AI Elements → assistant-ui → Prompt Kit |
| 大型公司的設計語言 | Fluent 2 → Spectrum → Carbon → Polaris → Primer |
